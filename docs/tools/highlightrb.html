Step-4

How do I use it?

1. First select the highlighting theme for the code you want to highlight. Use the "Previous" and "Next" buttons in the "Choice of theme:" section. Then select the markup language you want to display. Use the "Previous" and "Next" buttons in the "Language selection:" section.

2. In the second step, paste your code into the text box. If necessary, you can edit the code in the same field.

3. In the third step, you can see the result. This is how your highlighted code will look. Then with the "Copy code HTML" button, you can copy the resulting code to the clipboard, and then paste it in the right place on your HTML page. By clicking the "Ddownload css" button you can download the theme file in css format.

4. Next, include your theme css file between the <head> tags of your HTML page. Specify your path to the file.



<head>
  <link href="/assets/css/theme.css" rel="stylesheet" type="text/css">
</head>

5. Congratulations! That's all, your code will be highlighted according to the chosen theme. You don't need any javascript code!

ADDITION:If you are using Javascript, React, or JSX syntax, you will need code prepared differently. To do this At the third step, using t he "Copy code REACT" button, you can copy the received code to clipboard, and then paste it wherever you want in your javascript file. Here is an example of how to use the resulting code in the "render" ("React") function:
